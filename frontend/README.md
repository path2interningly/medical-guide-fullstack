# Medical Guide Frontend

React + Vite frontend for the Medical Reference Guide application.

## Setup

1. Copy `.env.example` to `.env` and update with your API and Supabase URLs.
2. `npm install`
3. `npm run dev`

Frontend runs on http://localhost:5173 by default.

## Development

- `npm run dev` - Start dev server with HMR
- `npm run build` - Build for production
- `npm run preview` - Preview production build
